Css Grundlegendes

  • Css "formt" visuelle Elemente.
  • Css selbst ist kein visuelles Element.
  • Css Vorlagen können gespeichert und per @import im Quellcode einer seite abgerufen werden.

Hier ein paar grundlegende Dinge direkt aus dem Quellcode dieser Seite

		@import "/Root Ordner/Unterordner 1/Unterordner 2/Css datei name.css";    */importiert den css Code einer css Datei um diese mittels des Klassennamens abzurufen*/
		
		.border1								/* Name der CSS class */
		{
			border:3px;						/* Dicke der Umrandung */
			display:inline-block;					/* Beschränkt größe der Umrandung auf Text */
			border-style:solid;					/* Art der Umrandung (hier:durchgezogen) */
			border-color:red;					/* Farbe der Umrandung */
			padding:1em;						/* Abstand der Umrandung zum Text */
		}
		
		a.Button								/*Initialisiert die Klasse Button
		
		.Buttoon a:link, a:visited					/*leitet die Definition von dem Elementen a:link und a:visited innerhalb der Klasse Button ein
		{
		
			background-color:red;				/* Hintergrund Farbe des Links während er "normal" angezeigt wird oder bereits besucht wurde */
			color:white;						/* Farbe des Textes */
			padding:15px 25px;					/* Abstand des Hitergrund randes zum Text erst vertikal , dann horizontal */
			text-align:center;					/* Text Ausrichtung */
			text-decoration:none;					/* entfernt den Unterstrich bei dem Link */
			display:inline-block;					/* größe des "buttons" wird an Textgröße angepasst */
		}
		
		.Button a:hover, a:active
		{
			background-color:black;

		}
		
		img
			{
				width:10%;					/* stellt die "standart" größe von Bildern auf 10% */
			}
			
		pre
			{
				font-family:Verdana; 				/*stellt die Schriftart von Text welche als pre ausgegeben werden auf Verdana*/
				font-size  :12px;	 			/*stellt die Schriftgröße von Text welcher als pre ausgegeben werden auf 12px*/
				
			}
